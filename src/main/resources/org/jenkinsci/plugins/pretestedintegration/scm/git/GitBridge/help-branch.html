What to specify

The branch name must match your integration branch name

Merge is performed the following way

Squash commit
            git checkout -B <integrationBranchName> <repoName>/<integrationBranchName>
            git merge --squash <devBranchFoundByGit>
            git commit -C <devBranchFoundByGit>
Accumulated commit
            git checkout -B <integrationBranchName> <repoName>/<integrationBranchName>
            git merge -m <commitMsg> <devBranchFoundByGit> --no-ff

When changes are pushed to the integration branch?

Changes are only ever pushed when the build results is SUCCESS

            git push <repoName> <branchName>
